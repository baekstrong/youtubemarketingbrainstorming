# 글쓰기 & 대본 프레임워크

## 제목 짓기 6가지 기법 (초사고글쓰기 기반)
1. **상식 파괴** — 상식적인 이미지를 나열하고, 반대되는 제목 짓기
2. **추상어 쓰지 않기** — 구체적인 주어, 숫자, 간접증명 사용
3. **자아 흠집내기** — 실행하면 해결되는 것에 대해 발끈하게 만들기
4. **금지 위협 (손실회피)** — 이걸 모르면 두려움을 느낄 문장
5. **권위 부여** — 전문가가 말하는 느낌
6. **뭐야 이게?** — 모를 만한 단어를 던져서 궁금하게 만들기

### 채널 규모별 제목 전략
- 채널 약할 때: **키워드 + 어그로(카피라이팅)**
- 채널 성장기: **어그로 + 키워드**
- 채널 초대형: **어그로(카피라이팅)**

## 썸네일 원칙
- **언매칭** — 제목과 썸네일 텍스트를 다르게
- **숫자 원칙** — 숫자로 구체성 부여
- **표본이론** — 타겟이 자신을 대입할 수 있게
- **비교 원칙** — 대비를 통한 시각적 임팩트
- **위협의 원칙** — 손실회피 본능 자극

## SEDA 원칙 (글쓰기 기본)
1. **S (Shortly)** — 짧게 쓰기
2. **E (Easily)** — 쉽게 쓰기 (쉬운 단어, 단순 문장, 예시)
3. **D (Divided)** — 문단 나누기
4. **A (Again)** — 다시 검토하기

## 깔때기 이론
- 앞부분은 쉽게 (많은 사람이 보도록)
- 뒤로 갈수록 어렵게 (필요한 사람만 보도록)

## 유튜브 대본 구조

### 인트로
```
1. 후킹 (초반 10초)
   - 독자가 원하는 결과 / 두려워하는 결과 제시
   - 제목 짓기 6가지 기법 활용
   - Yes세트, 이득, 위협, 스토리텔링(갈등→해소)

2. 공감 (마인드리딩)
   - Yes 세트로 방어기제 제거
   - "나도 그랬다" 공감대 형성

3. 권위 부여
   - 나의 경험, 능력, 후기, 권위자의 말
   - 근력학교 소개 + 경력 언급
   - 스토리텔링: 나도 같은 고통 → 어떻게 해결 → 더 나은 삶
```

### 본론
```
4. 주장 제시
5. 근거 1 + 사례
6. 근거 2 + 사례
7. 근거 3 + 사례 (선택)
```

### 아웃트로
```
8. 복습 (핵심 요약)
9. 행동 유도 (콜 투 액션)
   - "오늘부터 이것만 해보세요"
   - 뉴스레터/카페 안내
10. 기대 (다음 영상 예고)
11. 구독/좋아요 요청 (마지막에만)
```

## 본능 분석 체크리스트
콘텐츠 기획 전 반드시 확인:
```
✅ 심리분석: 시청자는 무엇을 원하는가?
✅ 상대방이 얻고 싶은 것: ?
✅ 예상되는 반박(의심): ?
✅ 반박 제거 전략: ?
✅ 콘텐츠 전체 전략: ?
```

## 톤 & 매너
- **대화체**: "~거든요", "~잖아요", "~인데요"
- **직장인 언어**: 야근, 회식, 출퇴근, 연차 자연스럽게 사용
- **겸손한 전문성**: "제가 다 아는 건 아니지만"
- **강압적 동기부여 지양**: "하세요!" 대신 현실적 공감 + 작은 실천
- **마무리 패턴**: "감사합니다. 백관장이었어요."

## 후킹 패턴 예시 (실제 사용된 것들)
- "사람들은 체력 딸리면, 그냥 유산소 운동이나 하라고 그러잖아요?"
- "양말을 앉아서 신으시나요?"
- "운동하면 더 피곤해져서 아예 포기하셨다고요? 사실 그거, 당신 잘못이 아닙니다."
- "체력 키우려고 머신 운동하고 있다면, 지금 당장 멈추세요"
