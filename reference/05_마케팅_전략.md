# 마케팅 전략 요약

## 비즈니스 퍼널 (9단계)

### 사전 단계 (잠재 고객)
| 단계 | 내용 |
|------|------|
| 1. 표적 고객 | 체력 저하를 느끼지만 운동 경험 없는 30대 사무직 직장인 |
| 2. 메시지 | "저질 체력 → 강철 체력, 주 2회 3시간 3가지 운동으로" |
| 3. 매체 | 유튜브, 인스타, 블로그, 스레드, 이메일(뉴스레터) |

### 진행 단계 (관심 고객)
| 단계 | 내용 |
|------|------|
| 4. 관심고객 포착 | 윤리적 뇌물 (뉴스레터, 구독자용 콘텐츠) |
| 5. 관심고객 육성 | 뉴스레터 (스티비) |
| 6. 판매 전환 | 신뢰 구축 → 100% 환불 보장 → 직접 체험 유도 |

### 후속 단계 (진짜 고객)
| 단계 | 내용 |
|------|------|
| 7. 최고의 경험 | 체력 향상 턴키 시스템, 수강생 관리 시스템 |
| 8. 고객생애가치 | 전자책, 온라인 강의, 현장 강의, 원데이 수업 확장 |
| 9. 추천 시스템 | 구축 예정 |

## 핵심 감정 키워드
- "일상을 되찾다" / "다시 살아나다"
- "하루가 2배로 늘어난 느낌"
- "양치질처럼 자연스럽게"
- "더 이상 ~가 핑계가 되지 않는"
- "화장실에 기어가던 → 55kg 들어올리는" (극적 대비)

## 직장인 페인포인트 (콘텐츠 소재용)
- 퇴근 후 소파에 쓰러짐
- 운동하면 오히려 더 피곤해짐
- 시간 없음 (야근, 회식)
- 헬스장 3개월 끊고 1번 감
- 양말을 앉아서 신음
- 의자에서 팔걸이 잡고 일어남
- 2층도 엘리베이터 탐
- 앉은지 1시간도 안돼 등이 굽음
- 팔다리는 얇은데 배만 나옴
