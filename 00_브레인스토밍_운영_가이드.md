# 유튜브 마케팅 팀 브레인스토밍 운영 가이드

## 채널 개요
- **채널 주제**: 직장인 체력 향상
- **타겟 시청자**: 25~45세 사무직 직장인, 만성 피로·체력 저하를 체감하는 사람
- **채널 톤**: 옆자리 동료가 알려주는 느낌, 현실 밀착, 작은 실천 유도

---

## 브레인스토밍 프로세스

### Phase 1: 주제 투입 & 리서치 (크리에이티브 디렉터 + 리서처)
```
1. 크리에이티브 디렉터가 주제를 프레이밍한다
   → "이 주제를 왜 다뤄야 하는가?" 한 문장 정의

2. 리서처가 사전 리서치 브리핑을 공유한다
   → 검색 트렌드, 경쟁 콘텐츠, 타겟 인사이트, 콘텐츠 갭
```

### Phase 2: 자유 발산 (전원 참여)
```
3. 전원이 자유롭게 아이디어를 낸다
   → 비판 금지, 양 우선
   → 각자의 전문 관점에서 아이디어 제시:
     - 크리에이티브 디렉터: 콘셉트·포맷
     - 리서처: 데이터 기반 제안
     - 라이터: 스토리·후킹 아이디어
     - 디자이너: 비주얼·연출 아이디어
     - 커뮤니티: 시청자 니즈 기반 제안
     - 커머스: 수익화 연결 아이디어
```

### Phase 3: 클러스터링 & 선별 (크리에이티브 디렉터 주도)
```
4. 유사 아이디어를 묶고 카테고리화한다
5. 각 카테고리에서 가장 강한 아이디어를 투표/선별한다
   → 선별 기준: 공감도 > 차별성 > 제작 가능성
```

### Phase 4: 구체화 (역할별 심화)
```
6. 라이터: 선별된 콘셉트의 스크립트 구조 설계
7. 디자이너: 썸네일 콘셉트 + 연출 방향 제안
8. 커뮤니티: 시청자 반응 예측 + 참여 유도 방안
9. 커머스: 구매 루트 + 수익화 전략 설계
10. 리서처: 최종 키워드·SEO 추천
```

### Phase 5: 최종 확정 (크리에이티브 디렉터)
```
11. 크리에이티브 디렉터가 모든 의견을 종합하여 최종 콘셉트 확정
12. 각 팀원에게 실행 태스크 배분
```

## 사용 방법

### 기본 사용
주제를 입력하면 6명의 에이전트가 순서대로 브레인스토밍에 참여합니다:

```
[사용자] 주제: "퇴근 후 운동 습관 만들기"

→ 크리에이티브 디렉터: 방향성 설정
→ 리서처: 데이터 브리핑
→ 전원: 자유 발산
→ 라이터: 스크립트 구조화
→ 디자이너: 비주얼 제안
→ 커뮤니티: 시청자 관점 피드백
→ 커머스: 수익화 연결
→ 크리에이티브 디렉터: 최종 정리
```

### 개별 에이전트 활용
특정 관점만 필요한 경우 개별 에이전트를 지정할 수 있습니다:

```
"리서처 관점에서 이 주제 분석해줘"
"커머스 매니저로서 이 영상의 수익화 방안 제안해줘"
"라이터로서 이 콘셉트의 스크립트 구조 잡아줘"
```
